1. What is DOM and what is the purpose of DOM in HTML?

The Document Object Model (DOM) is a programming interface for web documents. The purpose of the DOM in HTML is to provide a standard way for scripts and programs to access and manipulate web documents, allowing for the creation of dynamic and interactive web pages and web applications.

2. How to select an elements from dom?

There are five ways to select an element from dom?

  • 1. getElementsByTagName()
  • 2. getElementsByClassName()
  • 3. getElementById()
  • 4. querySelector()
  • 5. querySelectorAll()

3. What is event bubble in javaScript?

কোন একটা element এর মধ্যে EventHandler থাকে তাহলে সে নিজেকেও execute করে, তার সাথে parent ও যদি EventHandler থাকে তাহলে ওইটাকেও execute করে এবং ওই parent এর উপরেও যদি EventHandler থাকে তাহলে ওইটাকেও execute করে। মানে এক click এর জন্যে সবগুলাকে execute করে এইটাকে event bubble বলে। এখন এইটাকে বন্ধ করতে চাইলে তোমাকে stopPropagation() বা stopImmediatePropagation() দিতে হবে।
