1. What is DOM and what is the purpose of DOM in HTML?

The Document Object Model (DOM) is a programming interface for web documents. The purpose of the DOM in HTML is to provide a standard way for scripts and programs to access and manipulate web documents, allowing for the creation of dynamic and interactive web pages and web applications.
